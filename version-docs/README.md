## 版本编号说明

- **v0.1.0**：初始功能版本号，一次性交付完整核心功能
- **v0.1.1 ~ v0.1.6**：重新编排版本号，从大版本号调整为小版本号，依次迭代修复打包相关问题
- 后续版本将沿用 `v0.x.x` 的编排方式
