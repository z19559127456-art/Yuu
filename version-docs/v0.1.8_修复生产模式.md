# Yu v0.1.8 — 修复生产模式群聊记录清空

> **发布日期**：2026-05-07
> **git commit**：`<pending>`

---

## 版本概述

v0.1.8 是在 v0.1.6/v0.1.7 修复基础之上的**生产模式补全版本**。v0.1.6 只修复了源码层面，打包后的 `yu-backend.exe` 仍然运行旧代码。v0.1.8 重新构建了后端 exe 和完整安装包，确保生产模式下修复真正生效。

同时新增了数据库迁移系统的 v6 迁移（`group_id` 列），使迁移工具与代码变更保持同步。

---

## 修复内容

### 1. 生产模式后端代码生效（核心）

- **重新构建 `yu-backend.exe`**（PyInstaller），包含以下后端变更：
  - Message 模型 `group_id` 字段
  - `group_send` 消息持久化到数据库
  - `get_group_messages` WebSocket 处理
  - 数据库自动迁移（`ALTER TABLE` 添加 `group_id` 列）
- **重新打包 electron 安装包**（electron-builder），包含：
  - 最新前端代码（WebSocket 重连防护 + 群消息加载）
  - 最新后端 exe（消息持久化 + 历史查询）

### 2. 迁移系统 v6 迁移

- `scripts/migrate.py` 新增 v6 迁移
- 支持 `python -m scripts.migrate upgrade` 自动添加 `group_id` 列
- 包含 `idx_messages_group_id` 索引优化

### 3. 版本号统一

- `frontend/package.json`：0.1.7 → 0.1.8
- `backend/app/main.py`：0.1.7 → 0.1.8
- `backend/app/__init__.py`：0.1.7 → 0.1.8
- 安装包：`Yu-0.1.8-Setup-x64.exe`
