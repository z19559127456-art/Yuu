# Yu v0.1.9 — 多 Agent 协作增强

**发布日期**: 2026-05-08

## 新增功能

### 1. 自由对话模式（AutoGen 风格）
- Agent 在群聊中自由对话、协商、辩论，直至达成共识
- @提及优先级：被 @ 的 Agent 优先回复
- 共识检测：每 N 条消息自动检测是否达成一致
- 异议处理：共识达成后给异议者额外发言机会
- 防死锁：集成环路检测、预算控制、停顿监控
- 新增模块：`free_dialogue_manager.py`
- 前端：ChatHeader 模式切换下拉菜单、ChatInput 自适应消息类型

### 2. 任务智能拆分与派发（CrewAI 风格）
- 协调 Agent 将复杂目标拆解为子任务
- LLM 智能匹配：根据 Agent 角色/技能/工具/模型能力自动分配子任务
- 支持 4 种派发策略：CAPABILITY_SCORE / ROLE_MATCH / LOAD_BALANCED / ROUND_ROBIN
- 子任务按依赖关系并行执行
- 结果合并：所有子任务完成后自动合成最终输出
- 新增模块：`task_dispatcher.py`
- Orchestrator 适配：支持每个子任务在指定 Agent 上运行
- 前端：接入 PlanView + TaskProgress 组件，实时进度展示

### 3. Human-in-the-loop（人工审批）
- 4 个审批检查点：工具执行 / 计划审批 / 最终结果 / 高危操作
- 前端 ApprovalDialog：倒计时、批准/拒绝/修改、反馈输入
- WebSocket 实时推送审批请求
- 超时自动拒绝
- DB 持久化审批记录（`approval_records` 表）
- 新增模块：`approval_manager.py`

### 改进
- 群聊创建时可选 3 种模式：讨论 / 自由对话 / 任务
- i18n 中英文新增 30+ 字符
- ChatArea 条件渲染 PlanView + TaskProgress
- WS 消息类型新增 21 个
